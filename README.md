# Sorting_Algorithm_Visualization

This project is a Sorting Algorithm Visualizer built using Python and Pygame. It provides a graphical representation of sorting algorithms, helping users understand how sorting works in real-time.

## Features
- Supports Bubble Sort and Insertion Sort.
- Allows sorting in Ascending and Descending order.
 - Provides interactive controls:
        - R → Reset the list
        - SPACE → Start sorting
        - A → Sort in ascending order
        - D → Sort in descending order
        - I → Select Insertion Sort
        - B → Select Bubble Sort
- The visualization updates dynamically as the = algorithm progresses, making it an engaging way to learn sorting techniques!
